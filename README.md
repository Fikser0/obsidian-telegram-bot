# Server Assistant Bot

Telegram-бот на базі Python (aiogram 3) та Groq API (LLaMA 3) для керування сервером та ведення бази знань в Obsidian.

## Функціонал
* **Інтеграція з Obsidian:** Створення нотаток, ведення щоденника (Daily Notes) та TODO-листа.
* **Автоматична Git-синхронізація:** Бот самостійно робить commit та push у репозиторій з нотатками.
* **Розумний RAG-пошук:** Пошук інформації по існуючій базі знань Obsidian прямо в Telegram.
* **Smart Links & Media:** Парсинг веб-сторінок для красивого збереження посилань та підтримка завантаження фото/відео/документів.
* **Моніторинг сервера:** Перевірка метрик системи (RAM, температура, Uptime) та статусу Docker-контейнерів.
* **Голосове введення:** Розпізнавання голосових повідомлень через Whisper.
* **Безпека:** Фільтрація доступу за Telegram ID.
